## 📦 快速开始

### 1. 安装依赖

```bash
cd stock-backtest
pip3 install -r requirements.txt
```

依赖说明：
- `pandas` - 数据处理
- `numpy` - 数值计算
- `dataclasses` - 数据类支持（Python 3.6 需要）

### 2. 运行示例回测

```bash
# 双均线策略
python3 examples/run_backtest.py sma

# MACD 策略
python3 examples/run_backtest.py macd

# RSI 策略
python3 examples/run_backtest.py rsi
```

### 3. 查看报告

回测完成后，HTML 报告会自动生成在 `reports/backtest_report.html`，用浏览器打开即可查看：
- 权益曲线图
- 绩效指标
- 交易记录

## 📊 内置策略

### 1. 双均线策略 (SMACrossStrategy)

**原理**: 短期均线上穿长期均线时买入，下穿时卖出

**参数**:
- `short_window`: 短期均线周期（默认 10）
- `long_window`: 长期均线周期（默认 30）

**适用场景**: 趋势明显的市场

```python
from strategies.sma_cross import SMACrossStrategy
engine.add_strategy(SMACrossStrategy, short_window=10, long_window=30)
```

### 2. MACD 策略 (MACDStrategy)

**原理**: MACD 线上穿信号线时买入，下穿时卖出

**参数**:
- `fast_period`: 快线周期（默认 12）
- `slow_period`: 慢线周期（默认 26）
- `signal_period`: 信号线周期（默认 9）

**适用场景**: 趋势 + 震荡市场

```python
from strategies.macd import MACDStrategy
engine.add_strategy(MACDStrategy, fast_period=12, slow_period=26, signal_period=9)
```

### 3. RSI 策略 (RSIStrategy)

**原理**: RSI 低于超卖线时买入，高于超买线时卖出（均值回归）

**参数**:
- `period`: RSI 周期（默认 14）
- `oversold`: 超卖线（默认 30）
- `overbought`: 超买线（默认 70）

**适用场景**: 震荡市场

```python
from strategies.rsi import RSIStrategy
engine.add_strategy(RSIStrategy, period=14, oversold=30, overbought=70)
```

## 🔧 自定义策略

### 步骤 1: 继承基类

在 `strategies/` 目录下创建新文件，例如 `my_strategy.py`:

```python
import sys
import os
sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))

from strategies.base import BaseStrategy
from backtest.engine import Bar

class MyStrategy(BaseStrategy):
    """我的自定义策略"""
    
    def __init__(self, my_param=10, **params):
        super().__init__(**params)
        self.my_param = my_param
        self.prices = []
        self.in_position = False
    
    def on_init(self):
        print(f"策略初始化：my_param={self.my_param}")
    
    def on_bar(self, bar: Bar):
        self.prices.append(bar.close)
        
        # 数据不足时跳过
        if len(self.prices) < self.my_param:
            return
        
        # 你的交易逻辑
        avg_price = sum(self.prices[-self.my_param:]) / self.my_param
        
        if bar.close < avg_price * 0.95 and not self.in_position:
            # 价格低于均价 5%，买入
            self.buy()
            self.in_position = True
            print(f"买入 @ {bar.close:.2f}")
        
        elif bar.close > avg_price * 1.05 and self.in_position:
            # 价格高于均价 5%，卖出
            self.sell()
            self.in_position = False
            print(f"卖出 @ {bar.close:.2f}")
```

### 步骤 2: 运行回测

```python
from backtest.engine import BacktestEngine
from strategies.my_strategy import MyStrategy

engine = BacktestEngine(initial_cash=100000)
engine.add_strategy(MyStrategy, my_param=20)
engine.load_data('data/your_stock.csv')
results = engine.run()
engine.generate_report('reports/my_report.html')
```

## 📈 绩效指标说明

| 指标 | 说明 | 好坏标准 |
|------|------|----------|
| **总收益率** | 策略总收益百分比 | 越高越好 |
| **年化收益率** | 年化复利收益 | 越高越好 |
| **夏普比率** | 风险调整后收益 | >1 良好，>2 优秀 |
| **最大回撤** | 最大资金回撤 | 越低越好 |
| **胜率** | 盈利交易占比 | >50% 良好 |
| **盈亏比** | 平均盈利/平均亏损 | >1.5 良好 |

## 📁 数据格式

### CSV 数据格式

```csv
timestamp,open,high,low,close,volume
2022-01-01,10.00,10.50,9.80,10.20,1000000
2022-01-02,10.20,10.80,10.10,10.60,1200000
...
```

**必需字段**:
- `timestamp`: 时间戳（YYYY-MM-DD 或 ISO 格式）
- `open`: 开盘价
- `high`: 最高价
- `low`: 最低价
- `close`: 收盘价

**可选字段**:
- `volume`: 成交量

### 从真实数据源获取

#### 使用 AKShare（免费）

```python
import akshare as ak
import pandas as pd

# 获取 A 股数据
df = ak.stock_zh_a_hist(symbol="000001", period="daily", start_date="20220101", end_date="20231231")

# 转换为回测格式
df = df.rename(columns={
    '日期': 'timestamp',
    '开盘': 'open',
    '最高': 'high',
    '最低': 'low',
    '收盘': 'close',
    '成交量': 'volume'
})

# 保存
df.to_csv('data/000001.SZ.csv', index=False)
```

## 🚀 高级用法

### 多策略对比

```python
from backtest.engine import BacktestEngine
from strategies.sma_cross import SMACrossStrategy
from strategies.macd import MACDStrategy
from strategies.rsi import RSIStrategy

strategies = [
    ('SMA', SMACrossStrategy, {'short_window': 10, 'long_window': 30}),
    ('MACD', MACDStrategy, {'fast_period': 12, 'slow_period': 26}),
    ('RSI', RSIStrategy, {'period': 14, 'oversold': 30}),
]

results = {}
for name, strategy_class, params in strategies:
    engine = BacktestEngine(initial_cash=100000)
    engine.add_strategy(strategy_class, **params)
    engine.load_data('data/000001.SZ.csv')
    results[name] = engine.run()

# 对比结果
for name, result in results.items():
    print(f"{name}: 收益率={result.total_return:.2f}%, 夏普={result.sharpe_ratio:.2f}")
```

### 参数优化

```python
from itertools import product

best_return = -999
best_params = None

for short_win, long_win in product([5, 10, 15, 20], [20, 30, 40, 50]):
    if short_win >= long_win:
        continue
    
    engine = BacktestEngine(initial_cash=100000)
    engine.add_strategy(SMACrossStrategy, short_window=short_win, long_window=long_win)
    engine.load_data('data/000001.SZ.csv')
    result = engine.run()
    
    if result.total_return > best_return:
        best_return = result.total_return
        best_params = (short_win, long_win)

print(f"最优参数：短期={best_params[0]}, 长期={best_params[1]}, 收益={best_return:.2f}%")
```

## ⚠️ 风险提示

1. **回测≠实盘**: 回测结果不代表未来收益
2. **过拟合风险**: 参数优化可能导致过拟合
3. **交易成本**: 实盘手续费、滑点可能更高
4. **市场变化**: 策略可能在某些市场失效
5. **数据质量**: 确保使用准确的历史数据
